# Шрифты проекта

## Minecraft Font

Для полноценной работы интерфейса требуется шрифт **Minecraft**.

### Как установить:

1. **Скачайте шрифт Minecraft:**
   - Официальный: https://www.dafont.com/minecraft.font
   - Альтернатива: https://fontmeme.com/fonts/minecraft-font/

2. **Поместите файл шрифта в эту папку** (подойдёт имя `Minecraft.ttf` или `minecraft.ttf`)
   ```
   frontend/public/fonts/Minecraft.ttf
   ```

3. **Перезапустите dev-сервер**
   ```bash
   npm run dev
   ```

### Где используется:

- 🎮 Игровые элементы и счётчики
- 📊 Статистика в профиле
- 🏆 Достижения и бейджи
- 🎯 Специальные заголовки

### Альтернатива:

Если шрифт недоступен, система автоматически использует fallback:
```css
font-family: 'Minecraft', 'Orbitron', monospace;
```

---

**Примечание:** Шрифт Minecraft защищён авторским правом. Используйте только для некоммерческих/образовательных целей.
