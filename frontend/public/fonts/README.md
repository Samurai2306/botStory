# Шрифты

По умолчанию в проекте используется пиксельный шрифт **Press Start 2P** (подключён из Google Fonts в `index.css`). Чтобы использовать свой **Minecraft.ttf**, положите файл в эту папку и добавьте в `index.css` в `@font-face` для `Minecraft` первым источником: `url('/fonts/Minecraft.ttf')`.
