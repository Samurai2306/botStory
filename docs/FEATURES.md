# Реализованные возможности

Актуальный список функционала платформы **Legend of B.O.T.** (образовательная платформа на языке Кумир).

## ✅ Phase 1: Инфраструктура и аутентификация

### Инфраструктура
- ✅ Docker Compose конфигурация для всех сервисов
- ✅ FastAPI backend с асинхронной поддержкой
- ✅ React + TypeScript + Vite frontend
- ✅ PostgreSQL база данных
- ✅ Redis для кэширования и реалтайм-функций
- ✅ Alembic для миграций БД

### Аутентификация
- ✅ JWT-based аутентификация
- ✅ Регистрация пользователей
- ✅ Вход/выход из системы
- ✅ Защита эндпоинтов по ролям
- ✅ Три роли: Guest, User, Admin

## ✅ Phase 2: Лендинг и главная страница

### Публичные страницы
- ✅ Лендинг с описанием проекта
- ✅ Список возможностей
- ✅ Лента новостей
- ✅ Страницы Login/Register
- ✅ Адаптивный дизайн

### Хаб уровней
- ✅ Сетка уровней с карточками и анимированным 3D-кубом
- ✅ Индикатор прохождения на гранях куба (галочка/крестик, шрифт Minecraft)
- ✅ Отображение сложности (звёздочки)
- ✅ Фильтрация по статусу прохождения (все / пройденные / в процессе)
- ✅ Переход к брифингу уровня

## ✅ Phase 3: Игровой движок

## 🎯 Ключевые технологии

### Backend
- FastAPI 0.109
- SQLAlchemy 2.0 (ORM)
- Alembic (миграции)
- PostgreSQL 15
- Redis 7
- JWT аутентификация

### Frontend
- React 18
- TypeScript 5
- Vite 5
- Zustand (state management)
- Axios (API client)
- CodeMirror 6 (IDE)
- Canvas API (изометрия)

### DevOps
- Docker & Docker Compose
- Nginx
- Let's Encrypt (SSL)
